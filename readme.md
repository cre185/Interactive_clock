### 实现思路   
#### 整体布局  
为了建立一个能够统一给三个应用使用的布局，将整个页面规划为了四个区域：  
* 导航栏：负责页面间的切换，独立于各页面具体的逻辑；  
* 左侧边栏：作为陈列信息的空间，显示详细信息；
* 右侧边栏：和左侧边栏相似，在之后的开发过程中逐渐被用于展示多项元素；
* 中部空间：用于展示时钟/时间，包含页面的核心部分。
